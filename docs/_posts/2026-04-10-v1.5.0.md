## Better support for YouTube's updated card layout

This release improves button rendering on modern YouTube card components, including newer thumbnail view models used across Home and Results pages.

## New undo flow and icon state after adding to Watch Later

We added an undo icon state after adding a video to the Watch Later playlist.

What changes after click:

- The button icon changes to a confirmed state
- The button color changes from red to green
- A circular 10-second countdown starts around the button

During this 10-second window, clicking the button again removes the video from Watch Later (undo action).
After the timer ends, the temporary undo state is cleared automatically, and you can still remove the video in the usual way.
