## Redesigned Website Interface

We refreshed the Watch Later Button website with a cleaner, more modern layout.  
The new structure improves visual consistency and makes key information easier to scan.

## Clearer Navigation

Navigation is now easier across Home, Privacy, Updates, and individual update posts.  
Active states and quick links make it clearer where you are and where to go next.

## Better Readability

We improved typography and spacing on content pages so section headings stand out and long text is easier to read.

## Updated Chrome Web Store Screenshots

We also updated screenshots on the Chrome Web Store listing to better reflect the current interface and user flow.

This release focuses on website UX improvements only. Extension functionality remains unchanged.
