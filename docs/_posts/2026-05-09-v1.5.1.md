## Main feature: Watch Later button is now available in YouTube Shorts

Version 1.5.1 brings full button support to YouTube Shorts, so you can add Shorts videos to Watch Later directly from the Shorts interface.

This update extends the one-click flow beyond classic watch pages and keeps behavior consistent between Watch and Shorts surfaces.

## More reliable button mounting on YouTube Watch and Shorts

Alongside Shorts support, this release improves mount reliability across dynamic YouTube layouts, especially during SPA navigation between videos and Shorts.

What was improved:

- Stronger fallback selectors for Watch and Shorts action areas
- Safer delayed backfill attempts when action containers appear late
- Better observer target selection for fast re-detection after navigation

## Cleaner undo behavior after navigation

The temporary undo state introduced in v1.5.0 is now reset more predictably when YouTube changes pages without a full reload.

This prevents stale visual states (such as active undo countdown style) from carrying over between unrelated videos.

## Better stability for card-level injection

We also improved card injection consistency across Home and Results, including modern lockup/card variants, while avoiding duplicate button mounts.

## Internal maintenance updates

- Improved re-initialization deduplication for burst navigation events
- Additional cleanup of retry/backfill timer handling
- Minor diagnostics improvements for debugging dynamic mount timing

Version 1.5.1 focuses on reliability and UI-state consistency without changing the core one-click workflow.
