## New languages now available in the extension popup

This update adds 7 new locales to improve accessibility for more users worldwide:

- Arabic (`ar`)
- Hindi (`hi`)
- Swedish (`sv`)
- Portuguese (Portugal) (`pt_PT`)
- Czech (`cs`)
- Greek (`el`)
- Romanian (`ro`)

## Why this update

These locales were prioritized based on 90-day user distribution data to cover large and medium user groups not previously localized.
